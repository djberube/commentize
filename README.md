## Usage
```
commentize -s 'Hello world'

```

### Output
```
################################################################################
##  Hello world                                                               ##
################################################################################
```

### Options
| Option  | Description       | Default value      |
| ------- | ----------------- | ------------------ |
| `-c`    | comment character | `#`                |
| `-w`    | width             | `80`               |
| `-s`    | text to insert    | (no default value) |
